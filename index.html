Commonly Used CSS Tags

Tag Name Description
<color> Font Color Sets the font color
<background-color> Background Color Sets the color of the space behind the element
<font-family> Font Type Sets the font type that will be visible
<text-align> Aligns Text Aligns given text to either right, left, center, or justify
<font-weight> Weight of Font Sets the weight of the font between normal, bold, or thin
<border-top> Borders Creates a border along the top of the given element
<padding> Padding around elements Creates space between the contents of the 'box' and its borders.
<margin> Space around elements in relation to others on the page Creates space between the paragraph/element and all other elements outside. Same shorthand as with padding.
<min/max width/height> Min/Max Width/Height Ensures a minimum width/height around an element
<overflow> Element Overflow Limit Controls what happens to elements that overflows outside its box
<visibility> Determines visibility of elements Can be set to 'hidden', 'visible', 'none', or 'collapse'.
<position> Position Can change the position of an element using one of the five: static, relative, absolute, fixed, sticky.
<z-index> Z-Index Controls how far back or forward an element should appear on the webpage when elements overlap. like the depth of an element.
<float> Floating Moves content all the way to either the right or left of the webpage. Width must be specified. Works for static or relative elements.
<clear> Clear Controls how elements should behave when they bump into each other on the page.
<top> Offsets Element Offsets elements by the designated number of pixels. In this case, along the top margin.

Notes

Tag Notes
<margin> MUST have a width set in order to apply. set the px for vertical spacing and then 'auto' to center within that space horizontally.Horizontal margins between two boxes are added together. Vertical margins are not added together, rather the larger of the two is applied and sets the distance between the two elements.
<padding> Can be specified based on the side you want to add space (left, right, top, bottom) or by adding the values going clockwise (starting with top). 3 values means left and right are equal. 2 values means the top/bottom are equal and the left/right are equal.
<padding> Can be specified based on the side you want to add space (left, right, top, bottom) or by adding the values going clockwise (starting with top). 3 values means left and right are equal. 2 values means the top/bottom are equal and the left/right are equal.
<overflow> Commonly used values: 'hidden', 'scroll', 'visible'. Property is set on a parent element to instruct how to handle child elements. Must have a set height or width.
<visibility> 'Hidden' value only hides the contents of the box, there will still be a space for it on the page. 'None' removes the box and elements from the page entirely.
<Box-Sizing> Setting it to 'border-box' keeps the height and width of the box fixed. Thickness and padding will fit inside the box so overall dimensions don't change.
<position> 'static'=default. Any value should be accompanied with the offset property to move it away from the default position (top, bottom, left, right) specified in pixels, em, or %. Relative: position an element relative to its default position Absolute: all other elements on the page will ignore the element/act like it's not present. can be pinned to any part of the web page, but the element will still move with the rest of the document when the page is scrolled. Fixed: set an element to a specific point on page regardless of user scrolling. Sticky: keeps element in the document flow as the user scrolls, but sticks it to a specific position as the page scrolls further.
<Z-Index> Uses integers. The higher the number, the further forward it will appear. Will not work with static elements. Ignored by 'static' elements.
<Clear> 'left': the left side of the element will not touch any other element within the same containing element. 'right': the right side of the element will not touch any other element within the same containing element. 'both': neither side of the element will touch any other element within the same containing element. 'none': the element can touch either side.
